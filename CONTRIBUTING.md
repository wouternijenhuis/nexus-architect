# Contributing to Nexus Architect

Thank you for your interest in contributing! Please see our [detailed contributing guide](docs/docs/dev/contributing.md) in the documentation.

## Quick Start

1. Fork and clone the repository
2. Run `npm run install:all` to install dependencies
3. Run `npm run dev` to start development servers
4. Make your changes
5. Run `npm test` to ensure tests pass
6. Submit a pull request

For more details, see the [full contributing guide](docs/docs/dev/contributing.md).
